### Algorithm_Study

------

- 기간 : 2022년 3월 ~
- 시간 : 매주 토요일 11시
- 언어 : `JAVA`



#### 스터디 규칙

------

1. 주마다 6문제를 선정해 문제 풀이 진행
2. 회의 시간에 각 한문제를 선정해 풀이 설명
   - 개념 + 알고리즘 + 풀이방식
3. 깃허브를 활용해 코드 리뷰 진행



#### 설명 방식

------

1. 적용 알고리즘에 대한 개념
2. 문제 풀이 접근 방식 설명
3. 기본 코드에 대한 설명
4. 개선한 코드에 대한 설명
5. 시간복잡도, 공간 복잡도 계산
6. 사용 라이브러리
7. 기타(문제 풀이에 어려웠던 점, 구현하고자 했는데 실패한 점)
